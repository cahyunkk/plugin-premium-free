# Oraxen - Lightweight & 1.21+ Optimized Edition

Plugin Minecraft premium/free yang telah dimodifikasi agar **lebih ringan**, **mendukung versi Minecraft 1.21+ (hingga versi terbaru)**, tanpa menghilangkan fitur bawaan sedikitpun.

## ✨ Fitur & Optimisasi
1. **Dukungan Versi Terbaru (1.21+)**: Diperbarui `api-version` ke `"1.21"` agar berjalan sempurna pada server Paper/Spigot/Folia versi 1.21 ke atas tanpa warning/deprecations.
2. **Lebih Ringan (Lightweight)**:
   - Menghapus modul telemetry/metrics bStats (`bstats`) untuk menghemat penggunaan RAM dan CPU background thread.
   - Mengoptimalkan ukuran file jar dan struktur resource.
3. **Fitur Utuh 100%**: Semua fitur custom items, blocks, glyphs, HUD, mechanics, dan integrasi plugin (PlaceholderAPI, ProtocolLib, dll.) tetap dipertahankan utuh.
